USARE questi comandi perchè vuol dire che ci sono delle commit create da GitHub Actions che non ci sono sul repository locale (ciò accade ad esempio quando uso un workflow su github che ogni tot minuti va a prendere dati da una pagina web; es file weather.json del mio sito MeteoUnipi):

git pull origin main --rebase
git push origin main

Explanation:
git pull origin main --rebase → brings down the new commits from GitHub (like the workflow’s updates to weather.json) and reapplies your changes on top.

git push origin main → now succeeds because you’re no longer behind.
